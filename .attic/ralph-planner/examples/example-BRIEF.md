# Project Brief: Ralph Planner Website

## One-line description
A comprehensive website showcasing the Ralph Planner plugin with documentation, examples, and interactive tutorials.

## Problem / Why this exists
Developers struggle to understand and adopt planning tools because they lack clear, executable workflows. Ralph Planner solves this by providing a goal-driven, iterative system that combines planning with execution.

## Success criteria (measurable)
- [ ] Complete plugin documentation with real examples
- [ ] Interactive tutorial showing goal-based orchestration
- [ ] Downloadable examples and templates
- [ ] Clear before/after workflow demonstrations
- [ ] Video walkthrough of key features

## Constraints
- Tech stack: HTML, CSS, JavaScript, React
- Non-negotiables: Mobile-responsive, accessible (WCAG 2.1 AA)
- Time / budget: 2 weeks sprint

## Out of scope
- User authentication system
- Payment processing
- Blog or CMS functionality
- Multi-language support
